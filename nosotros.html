PROYECTO GRUPO 7 - CODO A CODO

Este proyecto corresponde al Grupo 7 de la comisión 24142 del curso de Desarrollo Web Full Stack Node.js de Codo a Codo 2024.

CARACTERISTICAS

  • El sitio web consiste en una plataforma para una tienda de bebidas alcohólicas.
  • Desarrollado en html y css, se utiliza grid y flexbox para diversos elementos y secciones. Ademas, incorpora javascript para gestionar el menu desplegable en version movil y la exhibicion de productos
  • En el index.html se crea un slider dinamico: se crea un slider para cada categoria de bebida y dentro de esta, se seleccionan al azar entre las bebidas disponibles en stock
  • Los visitantes pueden ver los productos disponibles (link productos) los cuales se muestran a traves de un script de JS que recorre un array que agrupa las bebidas a traves de su tipo (ron, whisky, vino, etc).
  • Para cada categoria se muestran las bebidas disponibles, verificando previamente si estan en stock o no (en este caso, se muestra un cartel "sin stock"). Al clickear sobre una bebida se habilita una ventana modal que muestra la descripcion del producto, la cantidad en stock, el precio y permite seleccionar la cantidad deseada y añadirla al carrito de compras. Los productos elegidos y la cantidad se guardan en localstorage
  • El carrito de compras muestra un listado de las bebidas seleccionadas, la cantidad, su precio total y un totalizador general de toda la compra.
  • Se cuenta con un area de login y registracion, con un formulario diferente para cada accion. Al ingresar la contraseña que se usara para el registro, se valida su extension, existencia de numeros y coincidencia
  • Adicionalmente se incorporo un formulario de contacto.
